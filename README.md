## Como publicar no GitHub Pages

1. Extraia este ZIP.
2. Envie o arquivo `index.html` para a raiz do seu repositório.
3. Substitua o `index.html` antigo por este novo.
4. Aguarde alguns minutos para o GitHub Pages/Vercel atualizar.
5. Abra em aba anônima ou limpe o cache do navegador caso ainda apareça a versão antiga.
